JavaScript Events

onmousemove

This is an onmousemove event that creates circles as the the user moves the mouse on the screen. The circles have a linear transition where the width and the height of the circle grow to 80px, the border width to 5px, and the opacity of the circle becomes 0.

See the Pen Events by nando (@nando2018) on CodePen.

CSS Hover

The mouse hover events can be triggered using pure CSS. In this case we make sure that we add the :hover pseudo class to the button id. The second button uses the ::before pseudo class to add content that will grow along the width of the button. Both buttons use transitions as the mouse hovers over them.

See the Pen Mouse Hover #1 by nando (@nando2018) on CodePen.

onmousedown/onmouseup

We can also add functionality when the user clicks. The onmousedown event works when the user performs a click and onmouseup triggers when the user finishes the click. In this example, the button changes color when the user clicks on it. Notice that as long as the user holds the click, the button keeps the red color, otherwise it turns green.

See the Pen mouseDown/mouseUp by nando (@nando2018) on CodePen.

onkeyup/onkeydown

We can add events to control the pressing of a key and its release. The onkeydown will trigger a function to make the red bar grow to the right by pressing the right (→) arrow key. When the user releases the key, then another event will trigger in which the red bar will grow smaller. This is done with the onkeyup event.

See the Pen keyUp/keyDown by nando (@nando2018) on CodePen.

Touch Events

  • touchstart is fired when a touch point is placed on the touch surface.
  • touchmove is fired when a touch point is moved along the touch surface.
  • touchend is fired when a touch point is removed from the touch surface.
  • touchcancel is fired when a touch point has been disrupted in an implementation-specific manner (for example, too many touch points are created). See mozila.org

In this case we use touchmove and touchend to trigger the animation of the box changing colors and dimensions. Note - Works only on mobile devices, but mouse events are also activated to simulate the animation on the computer.

See the Pen ontouch Events by nando (@nando2018) on CodePen.
